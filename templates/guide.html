Установка плеера

Сайт работает в связке с плеером, для установки скачайте:
dgnmpv.zip - для higher-end видеокарт (Eg. GTX 1080, RTX 2070, RTX 3060, RX 590, 5700XT, 6600XT)
dgnmpv-low-end.zip - для lower-end видеокарт (Eg. GTX 980, GTX 1060, RX 570)
разархивируйте и запустите файл "RUN_ME_AS_ADMINISTRATOR.bat" ОТ ИМЕНИ АДМИНИСТРАТОРА


после установки папку можно удалить
